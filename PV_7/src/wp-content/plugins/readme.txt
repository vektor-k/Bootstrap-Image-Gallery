== Description ==

A plugin to use PlugRush on your wordpress enabled site. 
Easily add PlugRush widgets to your sidebar or theme, or enable popunder, mobile redirect and adblock detection. 
If you are not already a PlugRush publisher, it's free to register. 
PlugRush is a Traffic Trading Network that lets you trade and sell traffic on your website. 
It uses unique customizable widgets to display thumbs to other sites, and will easily let you get more traffic to your site, while also making you money.

== Installation ==

1. Upload the `plugrush` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the PlugRush settings page in the admin menu, and enter your PlugRush api key
